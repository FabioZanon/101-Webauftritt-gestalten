Das Internet

Heutzutage findet die Kommunikation mehrheitlich über das Internet statt. Aber wie ist dies möglich?

Wenn man einen Computer an das Internet anschliesst, wird er automatisch einen Teil des Internets. Denn es besteht grundsätzlich nur aus Computer und anderen Geräte auf der ganzen Welt, die miteinander verbunden sind. Dadurch wird die Kommunikation zwischen allen Geräten möglich. Sie kann jetzt zwar stattfinden, dies aber auch nur unter einem Umstand. Sie müssen alle die selbe Sprache besitzen, damit dies auch funktioniert. Dafür benutzen sie "IP" oder auch "Internet Protocol", dabei ist die IP sozusagen die Basis im Internet.

Was ist eine IP Adresse und DNS?

Eine IP Adresse gibt z.b einem Computer einen Namen für das Internet, also für die Kommunikation. Sie wird jedem mit dem Internet verbundenem Computer zugeteilt. Aufgrund der Knappheit der heutigen Generation, wird es immer üblicher eine IP Adresse zugeteilt zu bekommen. Zurzeit ist der aktuelle Stand die IPv4-Adresse. Demnächst folgt die IPv6. Da die Zahlenkombinationen der IP schwierig zu merken sind oder sich sogar jederzeit ändern können, wird der IP einen für das Internet einfacheren Namen zugeteilt. Die so genannte DNS (Domain Name System).

HTML

Bevor man irgendetwas in HTML machen kann, muss man das Grundgerüst einfügen. Für dies muss man "html:5" eingeben.

Oben im Bild wird das standart bzw. Grundgerüst dargestellt. Es besteht grundsätzlich aus einem "Head" und einem "Body". Im Head stehen eigentlich nur die Einstellungen, der Titel und das Einbinden von CSS. Im Body befindet sich alles, was sich auf einer Webiste befindet. Als erstes ändert man die Sprache auf Deutsch. Auf dem Bild ist sie in der Zeile zwei in Englisch, also "en" eingestellt. Die umstellung auf Deutsch bezweckt, dass man nicht jedesmal nach dem Aufrufen der Website bei der Übersetztung auf "Nein" klicken muss. Nachdem kann man den Titel (Zeile Sieben) beliebig ändern. Der Titel wird oben im Tab der Site ange zeigt.

In HTML werden die ungern genannten "Befehle" "Codes" genannt, und besteht aus einem oder mehreren Elementen. Zum Beispiel wird ein Text zwischen <p> und </p> geschrieben. Dieser Code nennt man ein "Element". Ein HTML-Element ist jedoch noch aufgeteilt in ein Start-und End-Tag. (Blau Markiert) Dazwischen befindet sich das Attribut, welches den Namen und den Wert enthält.

Vollständiges Element mit Start/End-Tag, Attribut, welches Namen und Wert enthält und einem Element-Inhalt:<a href="http://sbb.ch"> SBB <a/>

Hier steht <a> als Start-Tag, "href" als Name, der Link für den Wert, SBB für den Elementinhalt und </a>als End-Tag.
